LLOOMMPPAA is a tool for experimental design based on current knolwedge.

I am a DPhil candidate at the University of Oxford. I work between the Structural Genomics Consortium (SGC) and the Oxford Protein Informatics Group (OPIG) for short.

I am sponsored by GlaxoSmithKline and work with their Computational and Structural Chemistry department, developing tools to improve data analysis.

Please contact me at anthony.bradley@worc.ox.ac.uk, or view my webpage at Anthony Bradley

To use this tool online using publically available datasets just go ahead and click Get Started

You can't upload you're own data, you will have to download it for offline use if you want to do that

You can use this tool offline using your own data. For windows users simply download the windows distribution above and follow the Wiki

If you want to make your own server for site-wide use, download the source code and set up the server (using a Ubuntu virtual machine) as set out on this Wiki
